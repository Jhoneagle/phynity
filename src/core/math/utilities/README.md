# Utilities

This module will contain mathematical utility functions and helpers.

## Planned Components

- **Trigonometry:** sin, cos, tan with caching for repeated angles
- **Constants:** Pi, 2Pi, epsilon values, etc.
- **Conversions:** Degrees/radians, angle wrapping
- **Numerical:** Clamping, lerp, smoothstep, min/max
- **Small-angle approximations:** For numerical stability in physics
- **Floating-point:** Comparison with tolerances, NaN/infinity checks
- **Geometry:** Ray-plane intersection, sphere tests, etc.
